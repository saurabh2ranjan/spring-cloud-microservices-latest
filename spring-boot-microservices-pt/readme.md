# Steps 

This project is developed using Spring Boot Microservices architecture. In this example - we've used below technologies
- Spring Boot v2.4.2
- Spring Cloud API Gateway
- Spring Cloud Config
- Resilient4J
- Keyloak
- Hashicorp Vault
- Spring Security - OAuth2 Client and Resource Server
- Eureaka
